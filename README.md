# 🎮 Fun Button (v0.7.0)

一個充滿樂趣與療癒感的虛擬機械按鍵玩具，支援自定義音效、多玩具管理、多國語言切換以及精緻的 3D 視覺效果。

👉 **[立即體驗 Demo](https://truedano.github.io/funButton/)**

---

## ✨ 核心特色

- **🎹 擬真機械手感**：具備 3D 深度感、陰影變化與縮放回饋的鍵帽設計。
- **🎙️ 專業級錄音體驗 (v0.4.0 新增)**：
  - **即時視覺回饋**：錄音時具備動態音量能量條（Audio Level Meter）。
  - **預覽確認**：錄音後可先試聽、再決定儲存或捨棄，提供完整的播放進度條。
  - **智慧音訊處理**：自動切除開頭與末尾靜音（Silence Trimming），並進行音量正規化（Normalization），讓聲音點擊即響且音量統一。
  - **操作穩定化**：支援「點擊任意處停止」，並優化了麥克風設備調用與資源回收機制。
- **🔊 播放進度視覺化 (v0.4.0 新增)**：
  - **按鈕內進度條**：在編輯模式試聽時，進度會以漸變填充的方式直接顯示在播放按鈕內。
- **🌐 多國語言支援**：
  - **自動偵測**：根據瀏覽器語系自動顯示繁體中文或英文。
  - **手動切換**：隨時點擊 Globe 圖示切換界面語言。
  - **持久化**：自動記住使用者的語言偏好。
- **🔊 自定義音效**：
  - **錄音**：直接透過麥克風錄製您的專屬聲音。
  - **上傳**：支援上傳任何音訊檔案。
  - **合成音效**：內建模擬機械開關的物理 Click 聲。
- **🗃️ 多玩具管理**：
  - 可以建立多組獨立的「玩具」。
  - **全卡片選取**：更直覺的 UX，點擊清單卡片即可快速切換玩具。
- **📤 備份與分享**：
  - **全量備份**：一鍵匯出/匯入所有玩具。
  - **單一玩具分享**：支援匯出單一玩具 JSON 檔案（包含嵌入的 Base64 音訊資料）。
- **🎨 極致色彩自定義 (v0.5.0 重大更新)**：
  - **無限色盤**：不再受限於預設顏色，支援完整的 Color Picker，可自由挑選任何顏色。
  - **雙重自定義**：可獨立設定「外殼」與「按鍵」的顏色，創造專屬的個性配色。
  - **智能樣式生成**：系統會根據您選擇的顏色，自動計算陰影、高光與立體邊框，確保 3D 質感完美呈現。
  - **編輯體驗優化**：新增按鈕時預設為空並使用 Placeholder，減少手動刪除預設文字的操作。
- **🖼️ 圖片按鈕支援 (v0.7.0 新增)**：
  - **圖層疊加**：支援上傳圖片作為按鈕貼圖，結合 Multiply 混合模式保留 3D 光澤質感。
  - **自動優化**：前端自動壓縮與縮放圖片（300x300px），確保備份與載入效能。
  - **可讀性增強**：當按鈕有背景圖片時，自動為文字添加描邊陰影，確保清楚可見。
- **⚡ 高效編輯 (v0.6.0 新增)**：
  - **按鈕複製**：一鍵複製按鈕設定（包含顏色與音效），快速產生雙胞胎按鈕。
  - **直覺排序**：支援 Drag & Drop 拖曳功能。
    - **手機優化**：透過長按 (Long Press) 觸發排序，避免與頁面滾動衝突，提供原生的觸控手感。
    - **視覺引導**：拖曳時具備即時的視覺回饋與動畫。
- **🚀 精緻回饋**：
  - **Toast 通知系統**：使用現代化的磨砂玻璃效果提示，取代傳統彈窗。
- **💾 動態存檔**：所有配置均儲存在本地 `IndexedDB`，重新整理也不會遺失。

## 🛠️ 技術棧

- **Core**: React 19 + TypeScript
- **i18n**: i18next + react-i18next
- **Styling**: TailwindCSS 4 (Modern CSS-first approach)
- **Audio**: Web Audio API (Analyser, Buffer processing, normalization)
- **Storage**: idb-keyval (IndexedDB wrapper)
- **Build**: Vite 6
- **Deployment**: GitHub Actions

## 🏃 快速啟動

### 本地開發

1. 安裝套件：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 建立生產版本：
   ```bash
   npm run build
   ```

## 🛡️ 安全性說明

本專案為純前端應用，所有個人錄音與配置均儲存於您的瀏覽器本地空間，不會上傳至任何雲端伺服器。所有音效處理（剪輯、增益）均在客戶端完成。
